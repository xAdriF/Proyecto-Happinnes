## Introducción

Perdidos entre la inmensa marea de actividades culturales que existen 
cada día en nuestra ciudad, nos dimos cuenta de que hacía falta buscar 
un poco de orden y estructura en este inmenso oceano de posibilidades. 
Y así  surgio  HAPPINESS&Co; una agenda cultural que intenta recopilar 
y organizar todos los eventos culturales, de cualquier tipo, que tienen 
lugar en nuestra localidad e inmediaciones.  
Musica, teatro, exposiciones… Todo tiene cabida en nuestra agenda.  
Así  que… ¡Bienvenidos a **Happiness & Co.**.

## Instalación y Uso

### Requisitos Previos

- Navegador web actualizado (Chrome, Firefox, Edge)
- Editor de código (recomendado: VS Code)
- Git instalado

### Pasos de Instalación

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/xAdriF/Proyecto-Happinnes.git
   ```

2. **Acceder al directorio**
   ```bash
   cd ProyectoHappiness
   ```

3. **Abrir el proyecto**
   - Abrir `index.html` directamente en el navegador, o
   - Usar una extensión como "Live Server" en VS Code
